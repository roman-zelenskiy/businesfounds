Специализация: общие сведения

Наш мир становится все более цифровым, и руководителям компаний необходимо осмыслить огромный объем доступных сегодня данных. Чтобы принимать ключевые стратегические бизнес-решения и использовать данные в качестве конкурентного преимущества, крайне важно понимать, как извлекать из этих данных ключевые выводы. Специализация "Бизнес-аналитика" предназначена для начинающих менеджеров, старших менеджеров и руководителей компаний, которые хотят получить всесторонние знания в области бизнес-аналитики, объединяющие такие области, как наука о данных, аналитика и принятие бизнес-решений.

Специализация включает 6 курсов

Аналитика данных затрагивает почти все аспекты бизнеса. Для того чтобы предприятия могли извлечь выгоду из анализа данных, им нужны лидеры, которые понимают рабочий процесс бизнес-аналитики. Этот курс решает проблему нехватки человеческих навыков, предоставляя базовый набор навыков обработки данных, которые могут быть применены во многих сферах бизнеса. В этом курсе вы будете использовать язык анализа данных, R, для эффективной подготовки бизнес-данных для аналитических инструментов, таких как алгоритмы и визуализации. Очистка, преобразование, агрегирование и изменение формы данных - важный, но малозаметный этап в рабочем процессе бизнес-анализа. По мере освоения R для подготовки данных к анализу вы получите опыт использования RStudio, мощной интегрированной среды разработки (IDE), которая имеет множество встроенных функций, упрощающих кодирование в R. Изучая рабочий процесс бизнес-анализа, вы также рассмотрите взаимосвязь между принципами ведения бизнеса и анализом данных. В частности, вы узнаете, как делегирование полномочий, контроль и целесообразность влияют на способ обработки данных. Вам также будут представлены примеры бизнес-проблем, которые можно решить с помощью автоматизации и аналитики данных, и методы передачи результатов анализа данных, которые не требуют копирования и вставки с одной платформы на другую.

Этот курс знакомит студентов с наукой бизнес-аналитики, обращая внимание на искусное использование цифр в цифровом пространстве. Цель - дать бизнесменам и менеджерам основы, необходимые для применения аналитики данных в реальных задачах, с которыми они ежедневно сталкиваются в своей профессиональной жизни. Студенты научатся определять идеальный аналитический инструмент для своих конкретных нужд; понимать достоверные и надежные способы сбора, анализа и визуализации данных; а также использовать данные для принятия решений в своих агентствах, организациях или клиентах.

Этот курс представляет несколько инструментов для обработки бизнес-данных с целью получения действенных выводов. Самым важным инструментом является ум аналитика данных. Соответственно, в этом курсе вы узнаете, что значит обладать аналитическим мышлением. Вы также потренируетесь определять бизнес-проблемы, на которые можно ответить с помощью анализа данных. Затем вы познакомитесь с различными программными платформами для извлечения, преобразования и загрузки данных (ETL) в инструменты для проведения исследовательского анализа данных (EDA). В частности, вы будете практиковаться в использовании PowerBI, Alteryx и RStudio для проведения процессов ETL и EDA. Результаты обучения по данному курсу включают: 1. Развитие аналитического мышления для решения бизнес-задач. 2. Способность оценивать ценность наборов данных для решения бизнес-задач с помощью сводной статистики и визуализации данных. 3. Умение грамотно работать с программным обеспечением для бизнес-аналитики для исследовательского анализа данных.

Одним из наиболее интересных аспектов бизнес-аналитики является поиск закономерностей в данных с помощью алгоритмов машинного обучения. В этом курсе вы получите концептуальные основы того, почему алгоритмы машинного обучения так важны, и как результирующие модели этих алгоритмов используются для поиска действенных идей, связанных с бизнес-проблемами. Некоторые алгоритмы используются для прогнозирования числовых результатов, другие - для прогнозирования классификации результатов. Другие алгоритмы используются для создания значимых групп из богатого набора данных. По окончании этого курса вы сможете описать, когда следует использовать каждый алгоритм. Вам также будет предоставлена возможность использовать R и RStudio для запуска этих алгоритмов и передачи результатов с помощью блокнотов R.

Этот курс знакомит студентов с маркетинговой аналитикой с помощью широкого спектра аналитических инструментов и подходов. Мы обсудим причинно-следственный анализ, анализ опросов с использованием регрессии, текстовый анализ (анализ настроений) и сетевой анализ. Цель курса - обеспечить основу, необходимую для принятия лучших маркетинговых решений путем анализа различных типов данных, связанных с удовлетворенностью клиентов.

Этот курс изучает применение бизнес-аналитики в бухгалтерском учете. Во-первых, в курсе представлен обзор технологических тем в бухгалтерском учете, в том числе технологический майнинг, блокчейн и приложения в аудите, налогообложении и обеспечении. Далее курс изучает визуализацию и базовую аналитику в аудите и контрольном тестировании с использованием R и Alteryx. Далее в курсе рассматривается применение анализа текста в бухгалтерском учете и проведение анализа текста с использованием R и RStudio. Наконец, в курсе рассматривается автоматизация роботизированных процессов в целом с использованием UiPath и ее применение в бухгалтерском учете.
